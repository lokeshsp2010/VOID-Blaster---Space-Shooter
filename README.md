# VOID-Blaster---Space-Shooter
<img width="636" height="761" alt="image" src="https://github.com/user-attachments/assets/97caa4d9-d918-4010-810a-7e272b05afe4" />

VOID Blaster is a retro-style space shooter game which is totally offline and is built from scratch using basic HTML code, CSS, and JavaScript. There is no server, and there is no game engine. Also, it can be used when you are bored, and you really need a game to pass the time. You can use that. As I said, other people can contribute to it by giving me reviews and by telling me which spots have the vulnerability and which can be improved to make the game much better. It can be deployed with a single click on the viewable demo link given below.
